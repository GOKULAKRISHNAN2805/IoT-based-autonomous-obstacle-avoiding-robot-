# IoT-Based Obstacle Avoiding Robot with Real-Time Monitoring

An intelligent IoT-based autonomous robotic system designed for obstacle detection, autonomous navigation, and real-time wireless monitoring using ESP32, Arduino, ultrasonic sensors, infrared sensors, and ESP32-CAM.

---

# 📌 Project Overview

This project presents a smart IoT-based obstacle avoiding robot capable of navigating autonomously while providing real-time video streaming and wireless monitoring. The system integrates embedded systems, robotics, IoT communication, and automation technologies into a single intelligent platform.

The robot continuously senses its environment using ultrasonic and infrared sensors. Based on real-time sensor data, the microcontroller processes information and controls the movement of the robot automatically to avoid collisions.

An ESP32-CAM module is integrated into the system to provide live video streaming over WiFi. Users can remotely monitor the robot’s surroundings through a web browser using the IP address generated by the ESP32 module.

# 🧠 Working Principle

The robot operates using a continuous loop mechanism based on the following process:

## 1. Obstacle Detection

The ultrasonic sensor continuously measures the distance between the robot and nearby obstacles.

IR sensors detect objects on the left and right sides of the robot.

## 2. Data Processing

The Arduino/ESP32 processes sensor data and compares distance values with predefined threshold conditions.

## 3. Decision Making

Based on obstacle conditions:
- If path is clear → robot moves forward
- If obstacle detected → robot stops
- If left side clear → robot turns left
- If right side clear → robot turns right
- If all directions blocked → robot reverses or stops

## 4. Motor Control

The L298N motor driver receives control signals from the microcontroller and controls motor movement.

## 5. Real-Time Monitoring

ESP32-CAM captures live video and streams it through WiFi using an internal web server.

📏 Ultrasonic Sensor Distance Formula

The ultrasonic sensor calculates distance using:

Distance = (Speed of Sound × Time Delay) / 2

This enables accurate obstacle detection and collision prevention.
